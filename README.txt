This program prompts the user to enter a number for the desired calculation using the following prompt: 
1) Reads in all 12 files from the previous year.
2) Stores the data in a multidimensional array.
3) Prompts the user for the desired calculation.
4) Performs the calculations and outputs the data to a file or the terminal. 
To compile the program, run the command:
gcc -g -Wall coffee_shop.o register.o -lm -o register 
To run the program, run the command:
.\coffee_shop   
and then enter a number when prompted
